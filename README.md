# Face Detection Webcam App

This project is a simple real-time face detection application using Python and OpenCV.  
The program opens the computer webcam, detects human faces using a Haar Cascade classifier, and draws a rectangle around every detected face.

It also displays the number of faces detected on the screen and allows the user to interact with the program using keyboard buttons.

## Features

- Opens the webcam in real time
- Detects faces using OpenCV Haar Cascade
- Draws a rectangle around each detected face
- Displays the number of detected faces
- Allows the user to change the rectangle color
- Allows the user to save a screenshot from the webcam
- Exits the program using the ESC key
